# Chatbot

A beginner-friendly terminal-based chatbot that responds to greetings, handles exit commands, and gives a default reply for unknown inputs.

## Features
- Responds to basic greetings  
- Understands exit commands (`bye`, `quit`, `exit`, `close`)  
- Simple fallback message  
- No external libraries needed  

## How to Run
1. Save the script as `chatbot.py`
2. Run it using:
   ```bash
   python chatbot.py
